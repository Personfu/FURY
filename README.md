## THE PREMIER OSINT & OFFENSIVE SECURITY DISTRIBUTION

FURY OSINT is a high-fidelity, enterprise-grade Linux distribution built upon the Kali Linux framework. It is engineered for intelligence operatives, senior security specialists, and future-forward researchers.

#### **Installer Images**

*Direct access to hardware | Customized FURY kernel | Minimal overhead*
Full control over hardware access (perfect for in-built Wi-Fi & GPU), enabling elite performance.

#### **Virtual Machines**

*Snapshots | Isolated environment | Pre-built for VMware & VirtualBox*
Ready-to-deploy environments without altering the host OS.

#### **ARM & Cloud**

*Raspberry Pi | PineBook | AWS | Azure | DigitalOcean*
Scale your intelligence gathering from leave-behind devices to high-end modern server clouds.

#### **Mobile (NetHunter)**

*Kali in your pocket | Android Integration | Kex support*

*Docker | LXC | Windows Subsystem for Linux*
